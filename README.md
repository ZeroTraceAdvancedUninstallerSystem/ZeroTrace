# ZeroTrace - Advanced Uninstaller System

**Author:** Mario B. | **License:** MIT | **.NET 8.0** | **Windows 10/11**

ZeroTrace is a professional open-source Windows application for complete software
removal. It detects and cleans residual files, folders, and registry entries.

## Features

- Deep Scan Engine with confidence-based scoring
- Safety Vault with automatic backup before deletion
- AES-256-GCM encrypted backups with SHA-256 verification
- Admin Dashboard with system health monitoring
- Auto-Update system for seamless upgrades
- Browser and temp file cleaning
- Startup program management
- File shredding (SSD/NVMe optimized)

## Build

```
dotnet build src/ZeroTrace/ZeroTrace.csproj -c Release
```

## Requirements

- Windows 10/11 (64-bit)
- .NET 8.0 Runtime
- Administrator privileges recommended
